American Airlines - January, 2020

This was the first job in which I was going to be able to use programming as a tool in the real world, to say I was excited for this opportunity is an understatement. During my time at American Airlines the main project in which I oversaw the development of was an airplane engine configuration application. Airplane engines as you can imagine are composed of many different parts which can possibly be interchanged with other engines. When a part of an engine needed to be replaced a team was needed to find which available part can be used. The team in charge of this configuration took about 4 hours to complete this task. Once I was done with the application they used it and where able to complete the configuration in about 5 minutes.
